As part of my Industrial Work Study (KKI) assignment at SMKN 1 Ciamis, i worked with [Anggi Prayoga](https://github.com/anggiprayoga9) to create a web-based admin panel. This system serves as the backend for the [Attendance Application](https://github.com/fauzidryy/presensi-app) developed by [Ahmad Fauzi Lubis](https://github.com/fauzidryy). We utilized pure JavaScript and jQuery for the front-end, Firebase for the database, and Mazer for the user interface. Available features include user account management, attendance history management, data import from Excel files, and student device management.
