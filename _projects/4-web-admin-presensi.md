Aplikasi ini dibuat untuk SMKN 1 Ciamis, sebagai bentuk tugas KKI (Kuliah Kerja Industri), yang dibuat oleh saya dan [Anggi Prayoga](https://github.com/Anggiprayoga9). Ini adalah proyek admin panel berbasis web untuk aplikasi Android [Aplikasi Presensi](https://github.com/Ahmadfzlbs/presensi-app) milik [Ahmad Fauzi Lubis](https://github.com/Ahmadfzlbs/presensi-app). Dibangun menggunakan vanilla JavaScript dan library [jQuery](https://jquery.com), [Firebase](https://firebase.google.com) untuk database, dan [Mazer](https://zuramai.github.io/mazer) untuk desain tampilan.
