During my Industrial Work Experience (KKI) at SMKN 1 Ciamis, [Anggi Prayoga](https://github.com/anggiprayoga9) and I developed a web-based guestbook application. This application is equipped with an admin panel for guest data management, visit records, and the creation of permission letters for teachers and students. This project was built using the CodeIgniter 3 framework.
